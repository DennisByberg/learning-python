# Variable Names

## Best Practices for Naming Variables

### Use Descriptive Names

```python
# Good - descriptive names
student_count = 1000
course_rating = 4.99
user_email = "john@example.com"

# Bad - unclear names
x = 1000
r = 4.99
e = "john@example.com"
```

### Use Lowercase Letters

```python
# Good
first_name = "Alice"
total_price = 99.99
is_active = True

# Avoid
FirstName = "Alice"
TOTAL_PRICE = 99.99
IsActive = True
```

### Use Spaces Around "="

```python
# Good - spaces around =
name = "Python"
age = 25
price = 19.99

# Bad - no spaces
name="Python"
age=25
price=19.99
```

## Key Points

- **Descriptive names** make code self-documenting
- **Lowercase with underscores** follows Python conventions
- **Spaces around =** improves code readability

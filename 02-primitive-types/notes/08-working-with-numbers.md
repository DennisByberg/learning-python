# Working With Numbers

## Built-in Functions

```python
print(round(2.9))    # 3 - Round to nearest integer
print(abs(-2.9))     # 2.9 - Absolute value
```

## Math Module

Import the **math module** for advanced functions:

```python
import math

# Rounding
print(math.ceil(2.2))   # 3 - Round up
print(math.floor(2.9))  # 2 - Round down

# Square root
print(math.sqrt(144))   # 12.0

# Constants
print(math.pi)  # 3.141592653589793
print(math.e)   # 2.718281828459045
```

## Key Points

- **Built-in**: `round()`, `abs()`
- **Import math** for advanced functions
- **`math.ceil()`** rounds up, **`math.floor()`** rounds down
- **`math.sqrt()`** for square root
- **Google "Python 3 math module"** for more
